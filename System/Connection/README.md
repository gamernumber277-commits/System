# Connection (1v1 Online)

## What this folder adds
- Realtime Socket.IO server for online 1v1 matchmaking.
- Browser client with movement + attack.

## Run
1. Open terminal at `c:\xampp\htdocs\System\Connection`
2. Install dependencies:
   `npm install`
3. Start server:
   `npm start`
4. Open two browser windows:
   `http://localhost:3000`

## Controls
- Move: `W A S D` or arrow keys
- Attack: `Space`

## Notes
- Home page can link to `http://localhost:3000` for online mode.
- This is a starter architecture; add anti-cheat and persistence for production.
